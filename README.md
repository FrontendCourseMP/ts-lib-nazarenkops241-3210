HTTP Client на TypeScript

Библиотека на TypeScript для работы с HTTP-запросами с поддержкой повторных попыток, типизацией и обработкой ошибок.

ВОЗМОЖНОСТИ

1 Методы: GET, POST, PUT, PATCH

2 Настраиваемое количество повторных попыток (retries)

3 Ошибки 400 и 500 выбрасываются как исключения

4 Автоматическая обработка JSON и заголовков

5 Поддержка query-параметров для GET-запросов

6 Полная типизация запросов и ответов

СТРУКТУРА

src/ — исходники проекта

httpClient.ts — класс HttpClient

types.ts — интерфейсы RequestOptions и HttpResponse

example.mts — пример использования

tests/httpClient.spec.ts — тесты для HttpClient

dist/ — скомпилированный JS

tsconfig.json — настройки TypeScript

package.json — скрипты и зависимости

ИСПОЛЬЗОВАНИЕ

Установить зависимости:

npm install


Скомпилировать TypeScript:

npx tsc


Запустить пример:

node dist/example.mjs


Запустить тесты:

npm run test


Тесты написаны с использованием Vitest и проверяют:

- Формирование URL с query-параметрами
- Отправку тела запроса для POST, PUT, PATCH
- Обработку ошибок 400 и 500
- Механизм повторных попыток при сбоях
- Валидацию количества повторных попыток (retries)
