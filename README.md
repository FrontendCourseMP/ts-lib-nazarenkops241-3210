# Библиотека HTTP-клиента на TypeScript

Простой TypeScript-класс для работы с HTTP-запросами с поддержкой методов `GET`, `POST`, `PUT` и `PATCH`.  

**Особенности:**
- Настраиваемое количество повторных попыток (`retries`)
- Ошибки 400 и 500 выбрасываются как исключения
- Автоматическая обработка JSON и заголовков
- Полная типизация запросов и ответов

**Структура проекта:**
- `src/httpClient.ts` — класс `HttpClient`
- `src/types.ts` — интерфейсы `RequestOptions` и `HttpResponse`
- `src/example.mts` — пример использования
- `dist/` — скомпилированный код
- `tsconfig.json` и `package.json` — конфигурации

Библиотека готова для расширения и использования в TypeScript-проектах.
